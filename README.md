# 📱 Mobile Price Range Prediction

## 🔍 Overview
This project predicts the price range of a mobile phone based on its features using Machine Learning.

The model classifies phones into:
- Low Cost
- Medium Cost
- High Cost
- Very High Cost

---

## ⚙️ Features Used
- Battery Power
- RAM
- Screen Size
- Camera (Front & Primary)
- Internal Memory
- Connectivity (WiFi, 4G, etc.)

---

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Flask (for web app)
